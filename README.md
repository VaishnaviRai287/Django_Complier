# APIForge

> An incremental API schema compiler and evolution engine for Django backends.

APIForge reads a custom `.api` DSL and compiles it into production-ready Django code — models, serializers, views, and URL routes — with built-in schema diffing and migration planning.

## Quick Start

source .venv/bin/activate
pip install -e ".[dev]"
apiforge --help
